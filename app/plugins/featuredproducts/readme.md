## Featured Products Plugin - Buysell 2.0

## Buysell Installation Instructions
	Steps to install Buysell:
	- Upload the featuredproducts script zip file into the Buysell script location of web server (Ex: /home/buysell/)
	- Unzip the file in the webserver
	- Give permissions (chmod –R 777 folder/) for the following folders & files
		- public/files
		- app/plugins/featuredproducts/lang
	- Execute the sql files in /app/plugins/featuredproducts/database/featuredproducts.sql

	That’s it! Featured Products Plugin - Buysell 2.0 is installed successfully.

	- Goto http://yourdomain.com/admin/featuredproducts/manage-featured-product-plans and add featured products plans

## Cron settings
	Cron setting in command prompt
	* * * * * curl --get http://yourdomain.com/featured-product-expiry-cron
	**Note: Please replace the yourdomain.com to your full URL where the script is installed.**
