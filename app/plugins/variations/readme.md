## Variations Plugin - Buysell 2.0

## Buysell Installation Instructions
	Steps to install Buysell:
	- Upload the variations script zip file into the Buysell script location of web server (Ex: /home/buysell/)
	- Unzip the file in the webserver
	- Give permissions (chmod –R 777 folder/) for the following folders & files
		- public/files
		- app/plugins/variations/lang
	- Execute the sql files in /app/plugins/variations/database/variations.sql

	That’s it! Variations Plugin - Buysell 2.0 is installed successfully.
